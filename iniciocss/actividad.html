1.1- ¿Qué significa CSS?
- CSS (Cascading Style Sheets) significa, hojas de estilo en cascada.

2 - ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?
- El CSS es un lenguaje de hojas de estilo, se utiliza para estilos de texto muy básicos,
para crear efectos de animación, como también convertir una columna de texto en una
composición con un área de contenido principal y una barra lateral.
- el programa que se utiliza son navegadores web como Chrome, Firefox, Safari, etc.

3- ¿A qué se llama regla de CSS?
- Se le llama reglas de CSS a un conjunto de propiedades y valores que se aplicarán a una
o varias etiquetas. Está compuesta por los elementos: Selector - Declaración - Propiedades
- Valor de la Propiedad.

4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista
descriptiva explicar sus elementos.

Selector
Selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos < p> ).
Para dar estilo a un elemento diferente, solo cambia el selector.
Declaración
Aplica las propiedades que querés dar al elemento seleccionado en el Selector.
En este caso hay una sola regla, pero puede haber más.
Propiedades
Es lo que te permite dar un estilo a un elemento HTML. En el ejemplo la propiedad es
color que le dará color a las letras de los párrafos. Siempre las propiedades se ubicarán
a la izquierda.
Valor de la Propiedad
las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor
se ubicará a la derecha.

5- ¿Qué es un selector en CSS?
- Un selector se utiliza para aplicar estilos a elementos HTML específicos. Esto permite un diseño
web preciso, especialmente util cuando un documento HTML contiene múltiples etiquetas iguales pero
con contenido diferente.

6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Selector de ID
Este selector lo aplicaremos a un único elemento HTML de nuestra página. Esto se debe a que el atributo
ID de HTML identifica a un elemento HTML por medio de un nombre. Por ejemplo:
< h1 id="titulo">Título Principal
Para aplicar CSS solamente a ID titulo, tendremos que seleccionar dicho ID utilizar el # seguido del nombre:

#titulo { color: violet; }
De la misma forma que el caso anterior, podemos seleccionar un elemento basado en el id. Por ejemplo:

a#titulo { color: red; }
En el ejemplo anterior se aplicará la propiedad color a la etiqueta a (anchor) que tenga el ID #titulo.
Selector de Clases
Los selectores de clases comienzan siempre por un . (punto). Los clientes web (navegadores) identifican
este punto y lo distinguen del resto de los selectores. Por ejemplo:

Texto de un párrafo

Para definir en HTML el uso de los selectores de clase, se utiliza el atributo class.
En css se aplica así:
.primer-parrafo {
font-size: 2em;
}
A todas las etiquetas que tengan la clase primer-parrafo, se aplicará un font-size (tamaño de letra) específico.

Las clases pueden ser utilizadas varias veces y en distintas etiquetas, lo que nos hace pensar que es necesario
ver cuales son la características que queremos aplicar a las etiquetas que sean comunes a varias.
A su vez, podemos seleccionar un conjunto de etiquetas que selecione todas las etiquetas que tengan una clase particular.

p.primer-parrafo {
color: #445544;
}
En el ejemplo anterior, seleccionamos todos los párrafos que tengan la clase primer-parrafo y le aplicamos un color de letra
especial.
Selector Descendente
Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo. Estos seectores siempre están
formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará
las propiedades de CSS. Por ejemplo:
p span {
color:green;
}
Selector por Etiquetas
Aplicamos propiedades CSS a una etiqueta. La misma se identifica sin utilizar <>. Por ejemplo.
p {
color:red;
}
Podemos, seleccionar varias etiquetas al mismo tiempo concatenándolas con el signo , (coma). Por ejemplo:
p,h1 {
color:red;
}
En el ejemplo anterior, tanto la p como la h1 tendrán de color de letra roja. Reordar que se aplicarán a todas las etiquetas
del documento HTML.
Selector Universal
Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el *.
* {
color:red;
}
Todos los elementos HTML tendrán un color de texto red.

7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

Paso 1
El navegador carga el HTML (por ejemplo, lo recibe de la red).
Paso 2
Convierte el HTML en un DOM (Modelo de objetos del documento). El DOM representa el documento en la memoria del ordenador.
Paso 3
Entonces, el navegador va a buscar la mayor parte de los recursos vinculados al documento HTML, como las imágenes y los
videos incrustados... ¡y también el CSS vinculado!
Paso 4
El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector.
Para cada tipo de selector que encuentre, calcula qué reglas deben aplicarse y a qué nodos en el DOM se les aplica el
estilo según corresponda (este paso intermedio se llama árbol de renderización).
Paso 5
El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
Paso 6
En la pantalla se muestra el aspecto visual de la página (esta etapa se llama pintura).

8- ¿Cuáles son las tres formas para vincular CSS con HTML?

  • Estilo en linea (Inline CSS) :
    • Se aplica directamente a un elemento con el atributo style .
    Es rápido para estilos específicos, pero es dificil de manejar.
  • Estilo interno (Internal CSS) :
    • Se define en una etiqueta < style> en el < head> . Aplica estilos a varios elementos
    en la misma página y no se utiliza en otras páginas.
  • Estilo externo (Externo CSS) :
    • Se usa un archivo CSS vinculado con < link> . Facilita el mantenimiento y es reutilizable, pero requiere
    carga adicional del archivo.
